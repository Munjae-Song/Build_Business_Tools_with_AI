# Build Business Tools with AI

Source code, notebooks, and sample data for *Build Business Tools with AI: A Problem-First Playbook for Domain Experts, Analysts, and Managers* (coming soon).

This book teaches non-developer business professionals how to build practical data solutions using Google Colab and AI. Every example in this repository runs in Google Colab — no local setup required.

## What's inside

| Folder | Description |
|--------|-------------|
| `000_interactive_dash_board` | Quality control dashboard — pulls 21 days of inspection data, calculates capability metrics, and displays interactive charts using Gradio |

More examples will be added as new chapters are published.

## How to use

1. Open the notebook link below.
2. Upload the sample data file to your Google Drive (or use the provided sample).
3. Click **Run All**.

## About the book

This book follows a simple sequence: define the problem clearly, find the right approach, then choose the solution.

It introduces two core methods — **3Q** for problem definition and **TCREI** for structuring AI requests — and applies them across real business domains including manufacturing, sales, logistics, and finance.
